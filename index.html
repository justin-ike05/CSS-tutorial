CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls how web elements, such as text, images, and layouts, appear on a webpage. CSS allows developers to separate the structure of a webpage (defined in HTML) from its visual style, making it easier to maintain and customize the appearance of multiple web pages.

Key features of CSS

  • Selectors: Used to target HTML elements to apply styles (e.g., p for paragraphs, .class for a class, #id for an element with an ID).
  • Properties and Values: CSS rules are composed of property-value pairs that define how an element should look (e.g., color: red;, font-size: 16px;).
  • Cascading: If multiple styles are applied to an element, CSS uses a "cascading" priority system to decide which style takes precedence (based on specificity, source order, and importance).
  • Responsive Design: CSS can be used to make web pages adjust to different screen sizes and devices via media queries.
